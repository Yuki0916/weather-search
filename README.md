#### 使用者能夠輸入特定的關鍵字，搜尋 GitHub 匹配到的 Repository Name 將傳回的結果以表格呈現，顯示以下欄位

- Repository Name
- Owner Name
- Created Date
- Updated Date
- Star Count
- Watchers

#### 每個欄位實作排序功能，且能進行升序、降序排序

#### 分頁功能，每頁最多十筆

#### 點擊表格上的其中一行，會同時顯示以下額外資訊:

- 列出該 Repository 的前十筆 Branch Name
- 列出該 Repository 的前十筆 Commit Message
- UI 在設計是跳出對話框。
